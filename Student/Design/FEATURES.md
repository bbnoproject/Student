# 기능 요구사항

## 1. 로그인 및 접근 제어

### 목적

내부 승인 사용자만 앱에 접근하고 학생 데이터에 권한 범위 내에서만 접근하도록 한다.

### 상세 기능

- Firebase 로그인
- 승인된 사용자 확인
- 역할별 권한 확인
- 권한 없는 사용자 접근 차단
- 세션 만료 시 재로그인
- 로그아웃

## 2. 학생 기본 정보 관리

### 목적

학생의 공통 프로필 정보를 등록하고, 이후 모든 기록의 기준점으로 사용한다.

### 상세 기능

- 학생 신규 등록
- 학생 정보 수정
- 학생 삭제 여부는 추후 정책 결정 후 반영
- 과정별 분류
- 기수별 분류
- 상태값 관리
- 학생 검색
- 학생 목록 정렬 및 필터

## 3. 모집단계 관리

### 목적

학생의 지원 및 선발 과정을 기록하고 비교 가능하도록 관리한다.

### 상세 기능

- 제출 서류 등록
- 기본정보 입력값 저장
- 자기소개 및 목표 텍스트 저장
- 면접 평가 항목별 점수 입력
- 면접 총점 자동 계산
- 면접 의견 입력
- 면접 진행자 기록
- 모집단계 상태 관리

### 분석 포인트

- 과정별 지원자 수
- 면접 평균 점수
- 합격/보류/불합격 비율
- 특정 평가 항목의 평균 점수 비교

## 4. 교과 과정 관리

### 4.1 출석 이슈

상세 기능

- 출석 이슈 기록 등록
- 이슈 유형 분류
- 사유 입력
- 기간별 출석 이슈 조회
- 누적 출석 이슈 수 확인

분석 포인트

- 지각/결석 빈도
- 특정 학생의 위험도
- 과정별 출석 이슈 비교

### 4.2 과제 제출 및 평가

상세 기능

- 과제 등록
- 제출 여부 기록
- 제출 시간 기록
- 평가 점수 입력
- 피드백 입력
- 학생별 제출률 계산

분석 포인트

- 학생별 제출률
- 과정별 평균 제출률
- 지연 제출 비율

### 4.3 프로젝트 기록

상세 기능

- 프로젝트명 기록
- 학생 역할 입력
- 일일 기록 입력
- 제출 여부 기록
- 시간 내 제출 여부 기록
- 운영진 메모 입력

분석 포인트

- 일일 기록 제출률
- 지연 제출 비율
- 역할별 참여 현황

### 4.4 면담 기록

상세 기능

- 면담 유형 기록
- 면담 내용 저장
- 면담 진행자 기록
- 후속 조치 여부 기록
- 경고 이력 관리

분석 포인트

- 학생별 면담 횟수
- 경고 학생 목록
- 최근 면담이 필요한 학생 탐지

### 4.5 취업 문서

상세 기능

- 자기소개서 내용 저장
- 제출 여부 기록
- 피드백 저장
- 최신 제출본 확인

분석 포인트

- 제출률
- 미제출 학생 목록
- 피드백 반복 횟수

## 5. 파일 업로드 기능

### 목적

운영진이 대량의 학생 데이터를 효율적으로 반영할 수 있도록 한다.

### 상세 기능

- CSV/XLSX 파일 업로드
- 업로드 템플릿 안내
- 컬럼 자동 매핑
- 수동 매핑 보정
- 데이터 미리보기
- 오류 행 표시
- 반영 결과 요약

## 6. 학생 상세 종합 보기

### 목적

한 명의 학생에 대한 모든 정보를 단일 화면에서 파악할 수 있도록 한다.

### 상세 기능

- 기본 프로필 요약
- 전체 커리큘럼 기반 학습 타임라인
- 모집단계 이력
- 출석 이슈 요약
- 과제 제출률 요약
- 프로젝트 기록 요약
- 최근 면담 이력
- 취업 문서 상태

### 학습 타임라인 세부 기능

- 학생 검색 후 상세 페이지 진입
- 시간표 기반 전체 과정 타임라인 표시
- 모집단계, 출결, 프로젝트, 면담 이벤트를 시간순으로 통합 표시
- 성향 평가 스냅샷 표시
- 이벤트 클릭 시 상세 내용 조회
- 위험 구간 강조 표시
- 프로젝트/면담/출결 필터 전환

## 7. 분석 및 리포트

### 목적

단순 저장을 넘어 운영에 필요한 인사이트를 제공한다.

### 상세 기능

- 과정별 학생 분포
- 단계별 진행 상태 분포
- 출석 이슈 랭킹
- 과제 제출률 리포트
- 프로젝트 기록 제출 리포트
- 면담/경고 리포트
- 취업 문서 제출 현황 리포트

### 위험 학생 탐지 예시

아래 조건을 조합해 위험 신호를 표시할 수 있다.

- 최근 30일 결석/지각 누적
- 과제 제출률 저조
- 프로젝트 일일 기록 누락 빈도 높음
- 경고 면담 이력 존재
- 취업 문서 장기 미제출

## 8. 설정 및 운영 기능

- 과정 목록 관리
- 면접 평가 항목 관리
- 출석 사유 코드 관리
- 사용자 권한 관리
- 업로드 템플릿 관리
- 테마 기본값 설정
- 세션 정책 설정

## 9. 보안 및 개인정보 보호 기능

- 민감 정보 접근 통제
- 권한 기반 메뉴/기능 노출 제어
- 중요 작업 이력 기록
- 파일 다운로드 제한 또는 권한 제어
- 장시간 미사용 시 세션 종료
- 민감 필드 마스킹 표시 검토

## 10. 로컬 앱 운영 요구사항

- Windows 설치형 앱으로 실행
- 공개 URL 없이 내부 배포
- 자동 업데이트는 후순위 검토
- 네트워크 연결이 있어야 Firebase 데이터와 동기화 가능
- 오프라인 영구 저장은 기본 지원하지 않음

## 11. 향후 확장 기능

- 학생 상태 자동 점수화
- 차트 고도화
- 알림 기능
- 외부 도구 연동
- 보고서 다운로드
- 다중 권한 체계 고도화
